DIMENSIONS: 


Based on the layout in "micfinder_final_layout.html" which implements the **Design B** specifications, here are the dimensions for the Sidebar (Drawer) components:

### **Mobile View (`< 768px`)**

* **Drawer Height (Max):** `70vh` (70% of the viewport height).
* **Drawer Height (Collapsed):** `0`.
* **Drawer Width:** `100%` (spans the full width of the screen).
* **Position:** Fixed to the bottom (`bottom: 0`, `left: 0`, `right: 0`).
* **Top Corners:** Rounded (`rounded-t-3xl`).

### **Desktop View (`>= 768px`)**

* **Drawer Height (Max):** `90vh` (90% of the viewport height).
* **Drawer Width:** `400px` (Fixed width).
* **Position:** Absolute positioning.
    * **Top:** `85px` from the top of the viewport.
    * **Left:** `16px` (`left-4`) from the left edge.
    * **Bottom:** Auto (height is determined by content up to max-height).
* **Corners:** Fully rounded (`rounded-3xl`).

### **Command Bar (Top Navigation)**

The command bar that controls the drawer has these dimensions:
* **Height:** `60px`.
* **Width (Mobile):** Spans full width with margins (`left-3`, `right-3`).
* **Width (Desktop):** `400px` (Matches the drawer width).
* **Position:** Absolute, `top-4`.

If you need to adjust these, look for the `#list-drawer` CSS block in the `<style>` section or the Tailwind classes in the HTML.
