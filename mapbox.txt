<script>
    // TO DO: Replace with your actual Access Token from Mapbox Dashboard
    mapboxgl.accessToken = 'YOUR_MAPBOX_ACCESS_TOKEN_HERE';

    // Initialize the Map
    const map = new mapboxgl.Map({
        container: 'map', // matches <div id="map">
        style: 'mapbox://styles/mapbox/streets-v12', // Standard Google Maps look
        center: [-74.006, 40.7128], // Starting position [lng, lat] (New York)

    // Initialize the Search Bar
    const geocoder = new MapboxGeocoder({
        accessToken: mapboxgl.accessToken,
        mapboxgl: mapboxgl,

        placeholder: 'Search venues, hosts...', // Custom text in bar
        collapsed: false // Keep the search bar open always
    });

    // Add Search Bar to Top-Left
    map.addControl(geocoder, 'top-left');

    // Add Zoom Controls (+/- buttons)
    map.addControl(new mapboxgl.NavigationControl(), 'bottom-right');

    // OPTIONAL: Capture the result for your app logic

Step 2: The Security Setup (Crucial)

Since you are paying if you go over the limit, you must stop other people from stealing your API key and using your quota.

Log in to Mapbox.com.

Scroll down to the "Access Tokens" section.

Click the "Create a token" button (It is better to make a new one specifically for this app rather than using the 'Default' one).

Name it: e.g., "Production Web App".

URL Restrictions (The Important Part):

In the URLs box, add the domains where your app will live.

For Testing: Add localhost and 127.0.0.1.

For Production: Add your actual website, e.g., https://www.my-app.com/.

Note: You can use wildcards. https://*.my-app.com/* covers all pages.

Click Create Token.

Copy this new token string (starts with pk.) and paste it into the code above where it says 'YOUR_MAPBOX_ACCESS_TOKEN_HERE'.
